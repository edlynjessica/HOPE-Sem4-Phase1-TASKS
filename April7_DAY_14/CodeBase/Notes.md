# HASHMAP 

* HashMap
* LinkedHashMap
* TreeHashMap

HashMap is a class which holds a key and a value as a parameter.

## HashMap methods:-

* get
* put

* ### getOrDefault
-> "used to retrieve the value associated with a specified key from the HashMap.
If the key is present in the map, it returns the corresponding value.
If the key is not present, it returns a default value that you can specify as a second parameter. 
If you do not provide a default value, it returns null."

* ### putIfAbsent 
-> "used to add a key-value pair to the HashMap only if the key is not already present in the map.
If the key is already present, the method does not update the value and 
returns the existing value associated with that key. 
If the key is not present, it adds the new key-value pair to the map and returns null."
